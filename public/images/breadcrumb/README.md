# Breadcrumb Background Images

## Struktur File
- `breadcrumb-bg.jpg` - Background utama untuk breadcrumb section
- `pattern.svg` - Pattern overlay untuk background

## Spesifikasi Gambar
Untuk `breadcrumb-bg.jpg`, gunakan gambar dengan spesifikasi:
- Ukuran: 1920px × 400px (atau rasio 4.8:1)
- Format: JPG/PNG dengan kualitas tinggi
- Warna: Gelap dengan kontras yang baik untuk teks putih
- Konten: Gambar abstrak, gradient, atau tekstur yang sesuai dengan tema website

## Cara Menambahkan Gambar
1. Letakkan gambar dengan nama `breadcrumb-bg.jpg` di folder ini
2. Gambar akan otomatis digunakan sebagai background breadcrumb
3. Untuk mengganti gambar, cukup ganti file `breadcrumb-bg.jpg`

## Contoh Gambar yang Cocok
- Gradient biru gelap ke hitam
- Tekstur abstrak dengan efek blur
- Pattern geometris minimalis
- Gambar dengan overlay gelap untuk keterbacaan teks

## Tips
- Gunakan gambar dengan resolusi tinggi untuk tampilan yang tajam di semua device
- Pastikan gambar cukup gelap agar teks putih mudah dibaca
- Kompres gambar untuk performa loading yang optimal
